# Contributing Guidelines

## Code Structure
1. Each problem should have solutions in all 5 languages
2. Follow the naming convention: `ProblemName.LanguageExtension`
3. Include algorithm explanation and complexity analysis

## Testing
- C#: Include xUnit tests
- C++: Include basic test cases in main
- JavaScript/TypeScript: Include Jest tests
- Dart: Include test files

## Pull Request Process
1. Update the README.md with new problems
2. Ensure all 5 language implementations exist
3. Update the progress tracking table
4. Add appropriate comments and documentation
